POI demo 4                   2013-11-05                 aok

This POI demo displays POI information on Google Maps.
POI information is queried from POI back-end. The format of POI information
is textual json data.

When a POI tag is clicked, the full data is shown as an infowindow on the map,
and the "3D" server is queried for associated data.
The response is logged to right panel.
